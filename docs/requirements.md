# System Requirements

## Server Requirements
- PHP 8.2+
- MySQL 5.7+ or MariaDB 10.2+
- Node.js 16+
- Composer 2+

## Browser Support
- Chrome 80+
- Firefox 75+
- Safari 13+
- Edge 80+

## Development Tools
- Git
- npm/yarn
- PHP extensions:
  - BCMath
  - Ctype
  - JSON
  - Mbstring
  - OpenSSL
  - PDO
  - Tokenizer
  - XML
